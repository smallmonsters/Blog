#### 三、使用异或交换两个数

``` JavaScript
let a = 5,
    b = 6;
a = a ^ b;
b = a ^ b; // b 等于 5
a = a ^ b; // a 等于 6
// 也可以使用解构
[a, b] = [b, a]
```
